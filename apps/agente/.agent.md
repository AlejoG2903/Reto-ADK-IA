# Agente de Cierre de Turno Clínico

```metadata
model: gemini-2.0-flash
description: Agente que genera cierres de turno clínico automáticamente
```

## Descripción

Este agente genera cierres de turno clínico automáticamente integrando datos de:
- Base de datos clínica
- Sistema de archivos
- APIs externas
- Análisis epidemiológico

## Capacidades

El agente puede:
- Obtener resumen epidemiológico
- Consultar datos por fecha y turno
- Acceder a información de pacientes
- Generar diagnósticos por consulta
- Compilar reportes de cierre de turno

## Herramientas

- **mcp_database**: Acceso a datos clínicos
- **mcp_filesystem**: Gestión de archivos
- **mcp_api**: Integración con APIs externas
- **mcp_analitica**: Análisis epidemiológico
